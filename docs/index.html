Breakfast recipes with beginner-friendly guidance

This site turns the repository's breakfast plan into a simple static website. Every recipe has its own page, and bean-based meals explain how to start from dry beans, chickpeas, or lentils instead of assuming canned shortcuts.

Dry bean help
Legume recipes include basic sorting, soaking, simmering, and storage guidance.

Using the site

  • Start with any day below and open the recipe page.
  • Read the Before you start section before you heat anything.
  • For bean, chickpea, or lentil breakfasts, use the pantry-stock section on that recipe page if you are beginning with dry legumes.

Quick dry-legume cheat sheet

  • Beans: sort, rinse, soak 8 to 12 hours if possible, then simmer until tender.
  • Chickpeas: soak overnight, then simmer until creamy inside.
  • Lentils: usually no soaking required; rinse and simmer until soft.
